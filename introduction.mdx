RALQ is a web-based chemistry education platform developed at Universidad Tecnológica del Sur de Veracruz (UTSV). The platform's name reflects its two core components: augmented reality (RA — *Realidad Aumentada*) and chemistry lab education (LQ — *Laboratorio Químico*). It combines traditional chemistry learning with augmented reality technology, letting students interact with 3D models of molecules and lab instruments directly from their browser or mobile device.

## Who RALQ is for

RALQ is built for chemistry students who want to:

- Understand lab instrument names and functions before stepping into a physical lab
- Visualize molecular structures in three dimensions to grasp bonding and geometry
- Study the periodic table with detailed element data including atomic mass, ionization energy, and electronegativity
- Self-assess their knowledge through interactive quizzes

<Note>
All in-app content is currently presented in Spanish, matching the instructional language at UTSV.
</Note>

## Technology overview

RALQ is a server-rendered PHP web application:

| Component | Technology |
|-----------|-----------|
| Backend | PHP 8.2 |
| Web server | Apache (mod_rewrite) |
| Database | MySQL (`registro_RALQ`) |
| Styling | Tailwind CSS |
| 3D/AR | Google Model Viewer (`@google/model-viewer`) |
| Containerization | Docker |
